# BATL project #
**BATL - Bratislava Tennis League** 
Tennis tournament and ranking management software for amateur tennis organization BATL.

## Users ##
- ADMIN - Super user
- ORGANIZER - Persons in charge of tournaments
- PLAYER - Participant in tournaments and rankings
- PUBLIC - A visitor accessing public-facing functionality

## Players ##
- person doesn't need to have an account to be a player and participate
- Organizer can create new player when they are registered into first tournament
