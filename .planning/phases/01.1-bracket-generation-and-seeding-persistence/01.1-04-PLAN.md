```javascript
/**
 * Bracket Persistence Service
 * Phase 01.1 - Bracket Generation and Seeding Persistence
 * API client for bracket generation, registration close, and slot swap endpoints.
 */
import apiClient from './apiClient';

/**
 * Close registration for a tournament
 * @param {string} tournamentId
 * @returns {Promise<{ registrationClosed: true }>}
 * @throws {ApiError} err.code: ALREADY_CLOSED | TOURNAMENT_NOT_FOUND
 */
export const closeRegistration = async (tournamentId) => {
  const response = await apiClient.patch(`/v1/tournaments/${tournamentId}/close-registration`);
  return response.data.data;
};

/**
 * Generate and persist a seeded bracket draw
 * @param {string} tournamentId
 * @param {Object} options
 * @param {string} [options.randomSeed] - Optional deterministic seed
 * @param {'PAIR_SCORE'|'AVERAGE_SCORE'} [options.doublesMethod='PAIR_SCORE']
 * @returns {Promise<{ bracketId, roundCount, matchCount }>}
 * @throws {ApiError} err.code: REGISTRATION_NOT_CLOSED | BRACKET_LOCKED | INSUFFICIENT_PLAYERS
 */
export const generateBracket = async (tournamentId, options = {}) => {
  const response = await apiClient.post(`/v1/tournaments/${tournamentId}/bracket`, options);
  return response.data.data;
};

/**
 * Batch swap player slots in the bracket
 * @param {string} tournamentId
 * @param {Array<{ matchId: string, field: 'player1Id'|'player2Id', newPlayerId: string }>} swaps
 * @returns {Promise<{ swapped: number }>}
 * @throws {ApiError} err.code: BYE_SLOT_NOT_SWAPPABLE | BRACKET_LOCKED
 */
export const swapSlots = async (tournamentId, swaps) => {
  const response = await apiClient.patch(`/v1/tournaments/${tournamentId}/bracket/slots`, { swaps });
  return response.data.data;
};
```

This file should contain no JSX, no React, no state — pure API calls only.

**State the component manages:**

- `registeredPlayers`: Array of `{ id, name }` — loaded once on mount, used for both State B list and State C dropdowns

**Loading registered players (load once, used across states):**

Use GET `/api/v1/registrations?tournamentId={id}` which returns `{ success: true, data: { registrations: [{ id, player: { id, name } }] } }`. Extract the player objects for a flat list:

```javascript
const [registeredPlayers, setRegisteredPlayers] = useState([]);
useEffect(() => {
  if (!tournament?.id) return;
  apiClient.get(`/v1/registrations?tournamentId=${tournament.id}`)
    .then(r => {
      const players = (r.data.data?.registrations || []).map(reg => reg.player);
      setRegisteredPlayers(players);
    })
    .catch(() => {}); // non-critical; dropdowns degrade gracefully if empty
}, [tournament?.id]);
```

**Rendering logic (conditional based on tournament state):**

State A — Registration still open (`tournament.registrationClosed === false`):
- Show registered player count (display only — no list needed)
- Show "Close Registration" button (primary, prominent)
- Clicking → calls closeRegistration(tournament.id), on success calls mutateTournament() to refresh

State B — Registration closed, no bracket yet (`tournament.registrationClosed === true` AND `structure?.brackets` is empty or missing):
- Show success badge "Registration closed"
- Show the finalized registered player list (per user decision in CONTEXT.md — locked): render `registeredPlayers` as a simple list (`<ul>`) or a count summary (e.g., "12 players registered") followed by the player names. Use the already-loaded `registeredPlayers` state — no second fetch needed.
- For DOUBLES categories (`tournament.category?.type === 'DOUBLES'`): show radio/select for seeding method (Pair score / Average player scores)
- Show "Generate Draw" button (primary, prominent)
- Clicking → calls generateBracket(tournament.id, { doublesMethod }), on success:
  - calls mutateFormatStructure() to refresh bracket view
  - calls mutateMatches() to refresh match data

State C — Bracket generated (`structure?.brackets` has entries):
- Show the KnockoutBracket component in edit mode (with slot dropdowns)
- Each non-BYE match slot (player1, player2) shows a `<Form.Select>` populated from `registeredPlayers`. Default selected value = current player id. `onChange` adds a swap to `pendingSwaps` state (collect locally, do not call API yet).
- BYE slots (`match.isBye === true`) show the player name as plain text — no dropdown.
- Show "Save Draw" button (primary) — disabled if `pendingSwaps.length === 0`
  - onClick: calls swapSlots(tournament.id, pendingSwaps), clears pendingSwaps, calls mutateMatches()
- Show "Regenerate Draw" button (outline-danger)
  - onClick: sets showRegenerateConfirm = true
- Regenerate confirmation modal: "Regenerating the draw will permanently delete the current bracket arrangement. Continue?" with Cancel and Regenerate buttons.
  - Confirm: calls generateBracket() again (regenerate = same endpoint, service deletes + recreates atomically), closes modal, calls mutateFormatStructure() and mutateMatches()

**Error display:** Show an Alert (variant="danger") when error state is set. Clear on next action start.
