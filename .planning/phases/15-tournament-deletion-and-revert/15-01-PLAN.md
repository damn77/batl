   - Inside `prisma.$transaction`:
     a. `tx.match.deleteMany({ where: { tournamentId: id } })`
     b. `tx.round.deleteMany({ where: { tournamentId: id } })`
     c. `tx.bracket.deleteMany({ where: { tournamentId: id } })`
     d. `tx.tournament.update({ where: { id }, data: { status: 'SCHEDULED', registrationClosed: false } })`
   - Return the updated tournament
