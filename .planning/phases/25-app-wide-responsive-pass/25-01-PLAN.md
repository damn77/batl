<task type="auto">
  <name>Task 1: Create app.css with design tokens, density overrides, tap targets, and overflow prevention</name>
  <files>frontend/src/app.css</files>
  <action>
Create a new file `frontend/src/app.css` with the following sections:

**1. Design Tokens (:root custom properties)**
Define spacing scale (tighter than Bootstrap defaults):
- --spacing-xs: 0.25rem (4px)
- --spacing-sm: 0.5rem (8px)
- --spacing-md: 0.75rem (12px)
- --spacing-lg: 1rem (16px)
- --spacing-xl: 1.5rem (24px)

Typography tokens:
- --font-size-sm: 0.8125rem (13px)
- --font-size-base: 0.9375rem (15px)
- --font-size-lg: 1.0625rem (17px)

Color tokens:
- --color-muted: #6c757d
- --color-surface: #f8f9fa
- --color-border: #dee2e6

**2. Bootstrap CSS variable overrides (tighter density globally)**
Override Bootstrap's built-in CSS custom properties in :root:
- --bs-card-spacer-y: 0.75rem (was 1rem)
- --bs-card-spacer-x: 0.75rem (was 1rem)
- --bs-card-cap-padding-y: 0.5rem
- --bs-card-cap-padding-x: 0.75rem

Apply tighter body font-size:
```css
body {
  font-size: var(--font-size-base);
  overflow-x: hidden;
}
```

Tighter card margins and form group spacing via class selectors:
```css
.card { margin-bottom: var(--spacing-md); }
.container, .container-fluid { max-width: 100%; }
```

**3. Global 44px tap targets (mobile only)**
At @media (max-width: 575.98px):
- `.btn` — min-height: 44px; (exclude .btn-group .btn with a more specific rule to reset)
- `.form-control, .form-select` — min-height: 44px; font-size: 1rem;
- `.list-group-item-action` — min-height: 44px; display: flex; align-items: center;
- `.nav-link` — min-height: 44px;
- `.page-link` — min-height: 44px;

**4. Overflow prevention**
```css
table { width: 100%; }
```

**5. Mobile density adjustments**
At @media (max-width: 575.98px):
- h1: font-size 1.5rem
- h2: font-size 1.25rem
- h3: font-size 1.1rem
- .container: padding-left and padding-right: var(--spacing-md) (12px, tighter than Bootstrap's 16px)
- .card-body: padding: var(--spacing-sm) (8px on mobile)

Use comments to clearly section the file. Do NOT use `!important` unless absolutely necessary to override Bootstrap. Prefer CSS specificity ordering (app.css loads after bootstrap.min.css).
  </action>

</task>
