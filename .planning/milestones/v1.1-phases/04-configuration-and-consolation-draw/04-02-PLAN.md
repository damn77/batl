    **Add new helper function `generateConsolationBracket(tx, tournamentId, mainBracketId, bracketSize, categoryType, matchGuarantee)`:**

    Mirror-draw structure rule: "loser of Main Match N vs loser of Main Match N+1" where N is 1-indexed.
    - The consolation bracket covers only Round 1 losers (the first elimination round).
    - Number of consolation Round 1 matches = bracketSize / 4 (half the main bracket Round 1 matches, since pairs of losers face each other).
    - Example: 8-player bracket (bracketSize=8): 4 main R1 matches → 2 consolation R1 matches. 16-player (bracketSize=16): 8 main R1 matches → 4 consolation R1 matches.
    - Consolation bracket has: 1 Round (Round 1 with empty slots — players TBD from main bracket results), and additional rounds up to the consolation final (totalConsolationRounds = log2(bracketSize/2)).

    Implementation:
    ```js
    async function generateConsolationBracket(tx, tournamentId, mainBracketId, bracketSize, categoryType, matchGuarantee) {
      // Create consolation Bracket record
      const consolationBracket = await tx.bracket.create({
        data: { tournamentId, bracketType: 'CONSOLATION', matchGuarantee }
      });

      // Consolation bracket size = bracketSize / 2 (half of main bracket players enter)
      const consolationSize = bracketSize / 2;
      const totalConsolationRounds = Math.log2(consolationSize);
      let matchNumber = 1000; // offset to avoid matchNumber collision with main bracket

      for (let roundNum = 1; roundNum <= totalConsolationRounds; roundNum++) {
        const round = await tx.round.create({
          data: { tournamentId, bracketId: consolationBracket.id, roundNumber: roundNum }
        });

        const matchesInRound = consolationSize / Math.pow(2, roundNum);
        for (let i = 0; i < matchesInRound; i++) {
          await tx.match.create({
            data: {
              tournamentId,
              bracketId: consolationBracket.id,
              roundId: round.id,
              matchNumber: matchNumber++,
              isBye: false,
              status: 'SCHEDULED',
              player1Id: null,
              player2Id: null
            }
          });
        }
      }

      return consolationBracket;
    }
    ```

    Note: All consolation match slots are null at draw time. Phase 5 (LIFE-01) will populate them as main bracket losers are determined. The match structure (N matches per round halving to the final) mirrors standard knockout logic.

    Note on matchNumber offset: Use 1000 + index to guarantee consolation match numbers don't collide with main bracket match numbers (which count from 1 up). Alternatively, read the highest existing matchNumber and continue from there — but the simple offset is sufficient for now.
