**Bug being investigated (NOT YET FIXED):**
- "I do not see a button to retrigger draw generation"

**Root cause analysis done:**
- The Regenerate Draw button exists in `BracketGenerationSection` State C (card header, `outline-danger` button)
- State C renders when `hasBracket = structure?.brackets && structure.brackets.length > 0`
- `BracketGenerationSection` only renders when `isOrganizerOrAdmin && tournament.status === 'SCHEDULED'` (FormatVisualization line 142) AND inner guard at line 163 returns null for IN_PROGRESS/COMPLETED/CANCELLED
- Most likely cause: after generating a bracket, `mutateFormatStructure()` is called but `hasBracket` stays false temporarily (SWR async update), leaving user stuck in State B which shows "Generate Draw" (not "Regenerate Draw")
- Fix approach: add local state `bracketGenerated` in BracketGenerationSection; set it to `true` after successful `generateBracket()` call; use `hasBracket || bracketGenerated` to determine State C — this immediately transitions to State C without waiting for SWR round-trip

**Files to modify for the fix:**
- `frontend/src/components/BracketGenerationSection.jsx` — add `bracketGenerated` state, set on generate success, use in hasBracket check

</remaining_work>
