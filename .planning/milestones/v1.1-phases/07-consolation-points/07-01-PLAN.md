Bracket/Round/Match schema:
```
Bracket: { id, tournamentId, bracketType: MAIN|CONSOLATION|PLACEMENT }
Round: { id, tournamentId, bracketId, roundNumber }
Match: { id, tournamentId, bracketId, roundId, matchNumber, player1Id, player2Id,
         pair1Id, pair2Id, isBye, status, result (JSON string) }
// Match.result JSON: { winner: "player1"|"player2", scores: [...] }
```

Consolation bracket structure (from bracketPersistenceService):
```
consolationSize = bracketSize / 2  (bracketSize = next power of 2 >= playerCount)
totalConsolationRounds = Math.log2(consolationSize)
matchNumbers start at 1000 (offset from main bracket)
```

       i. Players/pairs with zero consolation wins are NOT included (the function never creates an entry for them)
