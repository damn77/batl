Match data shape for doubles:
```javascript

  player1: null,
  player2: null
}
```

KnockoutBracket.jsx passes `isDoubles` prop to BracketMatch but NOT to MatchResultModal. The modal can detect doubles via presence of pair1/pair2 on the match object.
