## Accepted Design Decisions

1. **REVERT-04:** COMPLETED tournaments cannot be reverted — only deleted. Ranking recalculation is handled by DEL-05 on delete. This was a locked user decision during Phase 15 planning.
2. **ADMIN-02 ORGANIZER-as-player:** Plan 02 explicitly kept `/player/tournaments` route as `requiredRole="PLAYER"`. ORGANIZER self-registration happens via organizer management tools, not player registration page.
3. **UX-01/UX-02:** Phase 17 (Bracket View UX Fixes) removed from roadmap — user confirmed both issues are already working correctly.
