**2. Create `backend/src/api/groupStandingsController.js`:**

Three handlers:

`getStandings` — `async (req, res)`:
- Extract `groupId` from `req.params.groupId`
- Call `getGroupStandings(groupId)` from the service
- Return `{ success: true, data: result }`
- Catch errors: if group not found return 404 with `{ success: false, error: { code: 'GROUP_NOT_FOUND', message: 'Group not found' } }`

`saveOverride` — `async (req, res)`:
- Extract `groupId` from `req.params.groupId`, `positions` from `req.body.positions`
- Validate that all entityIds in positions belong to the group (query GroupParticipant where groupId matches, check playerId or pairId matches each entityId). Return 400 with code `INVALID_ENTITY` if any entity doesn't belong.
- Validate all position numbers are unique and contiguous. Return 400 with code `INVALID_POSITIONS` if duplicates.
- Find the latest `completedAt` among completed matches for this group: `prisma.match.findFirst({ where: { groupId, status: 'COMPLETED' }, orderBy: { completedAt: 'desc' }, select: { completedAt: true } })`. Use `new Date()` as fallback if no completed matches.
- Upsert `GroupTieResolution`: `prisma.groupTieResolution.upsert({ where: { groupId }, create: { groupId, positions: JSON.stringify(positions), resultSnapshotAt }, update: { positions: JSON.stringify(positions), resultSnapshotAt } })`
- Call `getGroupStandings(groupId)` and return updated standings
- Return `{ success: true, data: result }`

`deleteOverride` — `async (req, res)`:
- Extract `groupId` from `req.params.groupId`
- Delete: `prisma.groupTieResolution.deleteMany({ where: { groupId } })` (deleteMany instead of delete to avoid error if not exists)
- Call `getGroupStandings(groupId)` and return recalculated standings
- Return `{ success: true, data: result }`

**3. Create `backend/src/api/routes/groupStandingsRoutes.js`:**

```javascript
import express from 'express';
import { getStandings, saveOverride, deleteOverride } from '../groupStandingsController.js';
import { validateBody } from '../../middleware/validate.js';
import { isAuthenticated } from '../../middleware/auth.js';
import { authorize } from '../../middleware/authorize.js';
import { saveOverrideSchema } from '../validators/groupStandingsValidator.js';

const router = express.Router();

// GET /api/v1/tournaments/:tournamentId/groups/:groupId/standings
// Public endpoint — no authentication required
router.get('/:tournamentId/groups/:groupId/standings', getStandings);

// POST /api/v1/tournaments/:tournamentId/groups/:groupId/standings/override
// Protected — organizer/admin only
router.post(
  '/:tournamentId/groups/:groupId/standings/override',
  isAuthenticated,
  authorize('update', 'Tournament'),
  validateBody(saveOverrideSchema),
  saveOverride
);

// DELETE /api/v1/tournaments/:tournamentId/groups/:groupId/standings/override
// Protected — organizer/admin only
router.delete(
  '/:tournamentId/groups/:groupId/standings/override',
  isAuthenticated,
  authorize('update', 'Tournament'),
  deleteOverride
);

export default router;
```
