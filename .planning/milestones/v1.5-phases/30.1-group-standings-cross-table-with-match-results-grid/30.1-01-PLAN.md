**Props interface:**
```javascript
{
  entities,              // Array of { id, name } — group players or pairs
  matches,               // Array of match objects from useMatches()
  isDoubles,             // boolean — pair-based or player-based IDs
  scoringRules,          // { formatType, winningSets } — determines score display format
  highlightedMatchId,    // string|null — currently highlighted match ID (read from parent)
  onCellHover,           // (matchId) => void — called on mouseenter
  onCellLeave,           // () => void — called on mouseleave
  onCellClick            // (match, rowEntity, colEntity) => void — called on cell click
}
```

**Helper function — `getInitials(name)`:**
- Split name by whitespace. If 2+ parts: first char of first part + first char of last part, uppercase.
- Single-word name: first 2 chars uppercase.
- No name: return "?".
- For doubles pairs (name contains " / "): split on " / ", get initials of each, join with "/". Example: "Alice L / Bob M" -> "AL/BM".
- After computing all initials for the entities list, detect collisions and append numeric suffix: "AL", "AL2".

**Helper function — `getCellScore(match, rowEntityId, scoringRules)`:**
- If `!match || match.status !== 'COMPLETED'` return `null`.
- Parse `match.result` (may be string or object — try JSON.parse if string).
- Determine if row entity is player1: compare `rowEntityId` against `match.player1?.id || match.pair1?.id`.
- **Single-set / BIG_TIEBREAK detection:** `scoringRules?.formatType === 'BIG_TIEBREAK' || scoringRules?.winningSets === 1`.
  - If true: show game score from first set — `result.sets[0].player1Score` / `result.sets[0].player2Score`, swapped for perspective. Example: "6-4".
  - If false (multi-set): count sets won by each side from row player's perspective. Example: "2-1".
- Return the score string.

**Helper function — `getCellWinState(match, rowEntityId)`:**
- If `!match || match.status !== 'COMPLETED'` return `null`.
- Parse result, check `result.winner`. If `'PLAYER1'` and row entity is player1 -> `'win'`. If `'PLAYER2'` and row entity is player1 -> `'loss'`. Vice versa for other cases.
- Return `'win'` | `'loss'` | `null`.

**matchLookup — useMemo:**
```javascript
const matchLookup = useMemo(() => {
  const map = {};
  if (!matches) return map;
  for (const match of matches) {
    const id1 = isDoubles ? match.pair1?.id : match.player1?.id;
    const id2 = isDoubles ? match.pair2?.id : match.player2?.id;
    if (!id1 || !id2) continue;
    if (!map[id1]) map[id1] = {};
    if (!map[id2]) map[id2] = {};
    map[id1][id2] = match;
    map[id2][id1] = match;
  }
  return map;
}, [matches, isDoubles]);
```

**initialsMap — useMemo:**
Compute initials for each entity. Detect collisions: if two entities produce the same initials string, append "2", "3", etc. to subsequent ones. Store as `Map<entityId, initialsString>`.

**Render:**
- Wrap table in `<div className="table-responsive">` for mobile scroll.
- `<table className="table table-bordered cross-table">` with `<caption className="visually-hidden">Head-to-head match results grid. Rows are attacking player, columns are defending player.</caption>`.
- `<thead>`: First `<th>` empty (top-left corner). Then one `<th>` per entity showing their initials from `initialsMap`, using `className="text-center text-muted fw-bold cross-table-header"`.
- `<tbody>`: One `<tr>` per entity (row entity).
  - First `<td>`: entity full name (bold), same as standings Player column.
  - Then one `<td>` per column entity:
    - **Diagonal** (rowEntity.id === colEntity.id): `className="cross-table-diagonal"`, `aria-hidden="true"`, no content, no click/hover handlers, `style={{ cursor: 'default' }}`.
    - **Non-diagonal**: Look up `matchLookup[rowEntity.id]?.[colEntity.id]`. Compute score and win state.
      - `className` includes: `'cross-table-cell'`, `'text-center'`, win state class (`'text-success fw-bold'` for win, `'text-muted'` for loss or unplayed), highlight class (`'cross-highlight'` if `match && highlightedMatchId === match.id`).
      - `style={{ cursor: 'pointer' }}`.
      - `onClick={() => onCellClick(match, rowEntity, colEntity)` where match is `matchLookup[rowEntity.id]?.[colEntity.id]` (may be undefined for unscheduled — still clickable per locked decision, but only if match exists).
      - `onMouseEnter={() => match && onCellHover(match.id)}`, `onMouseLeave={() => onCellLeave()}`.
      - Content: score string if completed, `\u2013` (en-dash, not em-dash) if unplayed/scheduled. Per UI-SPEC Copywriting Contract.
      - `tabIndex={0}` for keyboard navigation. `onKeyDown` handler: if Enter or Space, call `onCellClick`.
      - `aria-label="{rowEntityName} vs {colEntityName}: {score}, {winState}"` for screen readers. For unplayed: `aria-label="{rowEntityName} vs {colEntityName}: not played"`.
